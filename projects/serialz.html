Serialz

Serialz is an interesting and odd project.

Serialz is a bot designed to serve in #serialz EFnet.

The current version of Serialz runs on a Microsoft Windows 2000 Machine running mIRC.

Due to the nature of this bot the scripts are currently private. However if you are looking for a copy of Serialz, send me an email and we can discuss it further.

Most of the scripts Serialz runs are based on older scripts (due to standardisation) such as ClownScript, DragNScript, Notepod and CowScript.

Serialz also has loaded MyIP, Google, Moo, Cmds, Wget scripts loaded.

If you are trying to find a Serial, try #serialz EFnet.
