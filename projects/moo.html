What does moo mean?
"moo" is often just a generic word used in place of "example" or "test", or even "foo". However of course it is the noise made by a cow.

What is moo script?
moo script was originally a script that displays your systems information via mIRC using moo.dll created by marky.
This script shows the stats of your system, including: your operating system, your uptime, your cpu, your memory usage, your resolution, your network and your hard drives, all in all, a useful script.

Example:

What is moo.dll?
moo.dll is an mIRC dll designed to return system information for mIRC scripts, that reports various statistics, made by marky.

Is moo.dll a virus or backdoor?
No, this is often a mis-conception as moo.dll is sometimes packaged up with trojans made with mIRC, such as GTbot, it is then used to retreive that systems information. It is NOT possible for moo.dll to be a backdoor or virus as it does not have the functionality to do so, all it does is return a strings of text.

What are moo.dll's functions?

The functions can be used within mIRC for example: //echo -a $dll(moo.dll,version,_) or //say $dll(moo.dll,version,_)

Function Description Example
$dll(moo.dll,version,_) Returns the version of moo.dll 4.0.4 build 1
$dll(moo.dll,osinfo,_) Returns the Operating System information Windows XP Professional, Service Pack 2 (5.1 - 2600)
$dll(moo.dll,uptime,_) Returns the uptime of your Operating System. 3d 9h 6m 38s
$dll(moo.dll,cpuinfo,_) Returns the CPU make, speed, cache and load. 1-AMD , 1799MHz, 256KB (0% Load)
$dll(moo.dll,meminfo,_) Returns the systems RAM usage. Usage: 470/512MB (91.80%)
$dll(moo.dll,screeninfo,_) Returns the systems screen resolution, bit rate and refresh rate. 1024x768 32bit 75Hz
$dll(moo.dll,gfxinfo,_) Returns the make and model of your graphics card. ATI Technologies Inc. RADEON 9600 Series AGP
$dll(moo.dll,connection,_) Returns details of the RAS dialup connection. -1
$dll(moo.dll,interfaceinfo,_) Returns the details of your network cards. #1 (SiS 900 PCI Fast Ethernet Adapter (100Mb/s) 2090.15MB In, 1906.62MB Out)
$dll(moo.dll,diskcapacity,_) Returns the free space and total space of your hard drives. C:\ (0.75GB Free, 55.89GB Total), D:\ (4.06GB Free, 111.78GB Total)
$dll(moo.dll,testing,_) Returns some CPU information that was under test. CPUID Processor Brand (where suppored, blank otherwise): (AMD Athlon(tm) XP 2200+)
$dll(moo.dll,mbm5info,1) If mbm5 is running it will return MBM5 details, otherwise it will error. error,Motherboard Monitor 5 is not loaded
