Updated
Incase you hadn't noticed, I have changed HM2K.org, i've cut it down to the bare essentials, to give people exactly what they want.

I decided to do this since I have updated a few of my scripts and released some new ones, yet found it difficult to announce, and organise.

I have also changed the site over from a table layout to a css layout, apparently its the "in thing" to do in the internet industry, so if it looks crap in your browser don't blame me, blame the industry.

The flash and gallery have been removed. The flash has moved over to ROFL.name anyway. And the gallery will probably return in private form.

I still have a few things to sort out with the news, but that can wait for now.

All the articles and scripts have been made a bit more openly avalible.

The most important thing for now is the files and the projects. Over the next few months I will be creating project pages for each of the projects listed, then using this I will be releasing the various information/files via these pages.

A few updated files that you might want to note include:
wget for mIRC - A very useful script, its meant to be a (simple) version of wget for unix based systems, you can simply type: /wget [url] and it will download the file to your mirc downloads folder.
lnks.us script - Since smlnk is no longer in use, I decided to change the script to work with lnks.us run by netmunky.
MyIP lookup script - whatismyip.com changed their html code *doh*, script was updated.
CJB.net Updater script for mIRC - Due to the myip lookup change, this script was updated.
Notepod for mIRC - enter a new notepod, its still in beta, but its the best and quickest notepod for mirc out there.
and most importantly...
moo script v3.0 *beta* (for mIRC v6.16 or above - no more moo.dll) - Thats right, no need for moo.dll anymore, but it is still in beta, as there are some bugs and it can be a bit slow at times.

(See, I have been busy. Let me know if I have missed anything.)

Links are no more, at least for now, however I will be writing a linkback monitoring script and prises will go to people sending me the most traffic.

Thanks and enjoy.
