]]> If you’re a geek like me or you’ve just  seen Iron Man 2 you’ll probably like the idea of having a custom motd display something cool when you login to your shell or terminal server.

Sure, you can use figlet to create a text banner, but how cool would it be if you could display your own logo instead. Right?

Can an image be converted into ASCII art?

Challenge accepted.

First of all, let’s get our logo image. We’re going to use the Gallush logo and to make it easier we’ll rehost the original image. Let’s go…

I started off by trying some online converts:

  • GlassGiant ASCII Art Generator
    • Normal output
  • mastervb.net ASCII Art
    • Highest quality output
  • picascii
    • Default output
  • DeGraeve.com IMG2TXT
    • Default output
  • lunatic.no Image to HTML/ASCII
    • Output
  • jennana.com JPG2ASCII
    • Result
    • Output
  • ConvertHub.com (as text/html)
    • Output

There’s a lot out there, which got me thinking…

What about software? Here’s the ASCII art converters I found:

  • aalib – outdated
  • jpeg2ascii – unfinished
  • jp2a – easy to install on FreeBSD
    • Output
  • libcaca

What about scripts? There must be a PHP, Perl or Python script out there…

  • JPG to ASCII Converter PHP snippet
  • Image2Char PHP Class
  • generating ASCII art from photographs in Python
  • pyascii
  • ppmascii (perl)
  • png2ascii.pl
  • Term::Caca – perl interface for libcaca (Colour AsCii Art library)
  • aalib-perl project

Then a brainwave, I want the image to fit a standard terminal screen (80 columns by 24 rows, according to putty’s default settings).

Since 1 pixel (usually) equates ASCII character, to make the ASCII art fit the screen I should resize the image from 128x128px to 24x24px.

Unfortunately at that size (24×24) it makes it almost impossible to work out what it’s meant to be, while at double the size (48×48) you can still make out all the defining features of the image. This is the resized image.

After doing this, my favourite so far was from “converthub.com”, which results in something a little like this which is exactly 24 lines so should fit your standard console screen.

]]> If you have landed yourself here chances are you already know why you want to do this, but for those who don’t here’s why:

When installing a new application, they sometimes require you to restart for them to work, often just restarting the Windows explorer shell will be sufficient for it to work as expected.

Normally most people just kill it via Task Manager, but this can have adverse effects on your system (such as data loss), so here’s how to do it properly:

  1. Go to Start -> “Turn Off Computer”
  2. At the “Turn off computer” window, hold down CTRL+ALT+SHIFT and click on “Cancel”.
    • When this is done, explorer should cleanly exit and your start menu will have gone.
  3. Hit CTRL+SHIFT+ESC to open Task Manager.
  4. Go to File -> “New Task (Run…)”.
  5. Enter: “explorer” in the “Open” box, and press OK.
    • If your start menu appears, explorer has restarted successfully, otherwise try again or reboot.

Enjoy!
